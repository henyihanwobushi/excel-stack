# excel-stack
Copy from input excel files and stack the sheets together.

## Input files
work book | work sheet
--|--
wb1 | 
| | ws1
| | ws2
wb2 | 
| | ws1
| | ws2

## Output  files
work book | work sheet
--|--
ws1 |
| | wb1
| | wb2
ws2 |
| | wb1
| | wb2

## Features
* copy worksheet between workbooks
* copy worksheet with format, cell merge
